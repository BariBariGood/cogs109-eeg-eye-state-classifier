# Presentation Materials

Study materials for the COGS 109 poster-session symposium. Use these alongside the poster
(`poster/variants/variant_1_polished_classic.pptx`) and the final report
(`report/final_report.md`).

## Files

| File | What it is | When to use it |
|---|---|---|
| `one_pager.md` | Printable cheat card with the elevator pitch + numbers + must-answer questions | Print, fold in pocket, glance at before the session |
| `study_guide.md` | Full ~6,300-word study guide: elevator pitch, deep dive, 30+ Q&A bank, panel-by-panel poster walkthrough | Read cover-to-cover the day before, drill the Q&A bank with a partner |
| `glossary_and_rationale.md` | ~6,400-word glossary of every poster term + every "what we did" paired with "why" + high-level implementation walkthrough | Reference doc — search for any term or design choice you blanked on |
| `anki_deck.csv` | 69 spaced-rep flashcards (Front,Back CSV importable into Anki) | Run a 10-min drill the morning of the session |

## How to import the Anki deck

1. Open Anki desktop.
2. File → Import → select `anki_deck.csv`.
3. Field separator: comma. Allow HTML in fields: off.
4. Pick or create a deck named "COGS 109 — EEG Eye-State".

## Recommended study schedule

- **3 days out** — read `study_guide.md` cover-to-cover once. Don't try to memorise.
- **2 days out** — read `glossary_and_rationale.md` cover-to-cover. Drill `anki_deck.csv` once.
- **1 day out** — practice the elevator pitch from `one_pager.md` aloud 5x. Run the Q&A bank with a partner.
- **Morning of** — drill `anki_deck.csv` again. Re-read the "Things to NOT say" section in `study_guide.md`.
- **5 min before walking up to the poster** — re-read the elevator pitch in `one_pager.md`.
