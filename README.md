# EEG Eye State Classifier

A COGS 109 project for classifying eye-open and eye-closed states from EEG data.

## Overview

This repository will track the data preparation, model training, and evaluation work for an EEG-based eye state classifier.

## Planned Work

- Load and inspect the EEG eye state dataset
- Preprocess features for modeling
- Train baseline classification models
- Evaluate model performance and summarize results
