# Poster

This directory holds the COGS 109 final poster for the EEG eye-state
classifier project.

## Files

- `poster.pptx` — source-of-truth artifact (48"×36" single-slide academic
  poster, generated by `scripts/build_poster.py`). Open directly, or
  import into Google Slides via *File → Import slides → upload*.
- `poster.md` — the same content, panel-by-panel, in Markdown. Useful as a
  fallback if the `.pptx` import loses formatting, or as raw source for
  pasting into a different template.
- `poster_preview.png` — a rendered preview (2048×1536 PNG) so the poster
  can be reviewed without opening LibreOffice or PowerPoint.

## Regenerate

From the repo root:

```bash
python scripts/build_poster.py
# or
make poster
```

The script is deterministic — running it twice produces a byte-identical
`poster.pptx`. The preview PNG is rendered from the pptx via LibreOffice
(`soffice --headless --convert-to png`); if `soffice` is not available
the script falls back to a PIL stub.

## What's on the poster

Layout: navy header bar with title + authors + UC San Diego mark, then
three columns:

| Column 1 (left)        | Column 2 (middle)             | Column 3 (right)            |
|------------------------|-------------------------------|-----------------------------|
| Abstract               | Methods                       | Results table               |
| Question / Hypothesis  | **Take-home (centrepiece)**   | Confusion matrices (Fig. 3) |
| Background + Fig. 1    | **Headline figure (Fig. 2)**  | Conclusions                 |
|                        |                               | Future work                 |
|                        |                               | References                  |

Embedded figures:

- `figures/08_label_autocorrelation.png` — motivates blocked CV.
- `figures/14_cv_comparison_three_way.png` — the headline 4-model × 3-CV
  comparison.
- `figures/15_confusion_matrices.png` — per-model holdout confusion
  matrices.

The headline number is **KNN (k=1) at 77.8% ± 2.7% stratified blocked CV
accuracy**, contrasted against the leaky-shuffled-CV figure of 97% for the
same model.
